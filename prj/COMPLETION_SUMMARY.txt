🔬 DETECTION ALGORITHM:
─────────────────────────────────────────────────────────────────────────────

1. Calibration Phase (First 100 frames):
   • Collect baseline EAR values
   • Compute individual user baseline
   • Adjust thresholds to 90% of baseline

2. Feature Extraction:
   • Compute EAR for left and right eyes separately
   • Average both eyes for combined EAR
   • Compute MAR from mouth landmarks
   • Maintain rolling history

3. Temporal Filtering:
   • DROWSY: 15+ consecutive frames below threshold
   • CRITICAL: 30+ consecutive frames significantly below
   • Prevents false positives from blinks

4. State Classification:
   • Active: Normal operations
   • Drowsy: Alert user with intervention
   • Critical: Emergency alert

5. Alert Triggering:
   • Cooldown: Prevent alert spam (5 seconds)
   • Duration: Alert lasts 3 seconds
   • Challenge: Math problem for intervention
